Mockup interaktif. Semua alur macOS dipetakan ke iPhone. Data dummy, tapi tombolnya benar-benar jalan.

Mulai dari onboarding. Tekan lanjut sampai masuk app.

Alur inti: tab Invois → tombol + → pilih klien → tambah item → Simpan → Bagikan.
Penyimpanan: iCloud Drive, satu vault, otomatis. Tidak ada pemilihan folder di iPhone.

Data kamu di iCloud

Invois menyimpan satu berkas vault di iCloud Drive → Invois. Mac dan iPhone membaca vault yang sama, jadi invoice yang kamu buat di sini langsung muncul di Mac.

Di Mac kamu bebas memilih folder mana pun. Di iPhone, iOS tidak mengizinkan itu tanpa merepotkan — jadi lokasinya kami tetapkan, dan tetap terbuka lewat app Files.
